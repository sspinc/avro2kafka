# CSV2Avro

Publish data from Avro files to Kafka in JSON format

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'avro2kafka'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install avro2kafka
